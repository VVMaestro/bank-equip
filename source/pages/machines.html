Банковское оборудование

Подходящее решение для центральных банков и крупных коммерческих банков, высокоэффективная система, которая использует передовые детекторы для удобной сортировки банкнот.

Высокоскоростная машина
для обработки банкнот (FS-2000)

FS-2000, созданная в соответствии со стандартами надежности, качества и инноваций компании Toshiba, является передовым решением для профессиональных расчётно-кассовых центров. FS-2000 настраивается с помощью дополнительного модуля обнаружения, онлайн-шредера и дополнительного накопителя / обвязочного устройства.

Машина для обработки банкнот (GB-5600)

Разработанная для обработки банкнот в расчётно-кассовых центрах центрального банка, GB-5600 может сортировать банкноты, скреплять вместе 100 годных банкнот и измельчать непригодные банкноты. GB-5600 эргономично укомплектована, что сокращает занимаемую площадь.

Машина для обработки банкнот (FS-810)

Разработанные специально для филиалов центральных банков, все функции, включая связывание банкнот в режиме рабочего времени и измельчение, эргономично укомплектованы, что способствует сокращению занимаемой площади.

Термоусадочная упаковочная
онлайн-машина (FS-P70 / P210)

Онлайн-машина для термоусадочной упаковки подключается к нашим системам обработки банкнот. FS-P70 / P210 может эффективно и надежно обернуть 10 стопок (или 1000 банкнот) с помощью прозрачной пластиковой пленки.
